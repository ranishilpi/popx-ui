# PopX UI

This project is a React implementation of the PopX mobile UI provided as an internship assignment.

## Tech Stack

* React (Vite)
* React Router
* React Icons
* CSS

## Features

* Welcome screen
* Login screen
* Register screen
* Profile / Account settings screen
* Bottom navigation with page indicator (1 of 4)

## Installation

Clone the repository:

git clone https://github.com/ranishilpi/popx-ui.git

Go to the project folder:

cd popx-ui

Install dependencies:

npm install

Run the development server:

npm run dev

## Build

To create a production build:

npm run build
